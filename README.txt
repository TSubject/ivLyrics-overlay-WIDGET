===========================================
    MusicWidget v1.0 Beta
    음악 위젯 + GIF 애니메이션
===========================================

[설치 방법]
1. 폴더 전체를 원하는 위치에 복사
2. MusicWidget.exe 실행

[기본 사용법]
- 음악 재생 시 자동으로 정보 표시
- GIF가 함께 재생됨
- 드래그로 위치 이동 가능

[GIF 추가]
- assets 폴더에 GIF 파일 복사
- 자동으로 화면에 표시됨

[게임 모드]
- 전체화면 게임 실행 시 자동으로 클릭 투과
- 게임 종료/Alt+Tab 시 자동 해제
- 설정 버튼(⚙️)은 항상 클릭 가능

[단축키]
- Ctrl+Shift+C : 클릭 투과 수동 토글
- ESC : 최소화
- Shift+마우스휠 : GIF 크기 조절

[GIF 조작]
- 클릭 : 해당 GIF를 맨 위로
- 드래그 : 위치 이동
- 모서리 드래그 : 크기 조절
- 우클릭 : GIF 숨기기

[설정 메뉴]
설정 버튼(⚙️) 클릭으로 접근:
- Show GIFs : GIF 표시/숨기기
- GIF Speed : 속도 조절
- Auto Start : 윈도우 시작 시 자동 실행
- Click-through : 클릭 투과 모드
- Auto Game Mode : 자동 게임 모드

[시스템 요구사항]
- Windows 10/11
- .NET Framework 4.5 이상

[알려진 이슈]
- 멀티 모니터: 기본 모니터만 게임 감지
- 창 모드 게임: 전체화면만 감지됨
